HTML Lists

HTML provides a way to organize and present information using lists. Lists allow you to display information in a structured and easy-to-read format. There are three main types of lists in HTML:

1. Unordered Lists (UL)

An unordered list is used to represent a list of items that have no specific order. Each item in the list is preceded by a bullet point (or other list marker). To create an unordered list, use the <ul> element and enclose list items with <li> elements:
