HTML Tables

HTML tables are a fundamental part of web page layout and are used to organize and display data in a structured format. Tables consist of rows and columns, where each cell in the table can contain text, images, links, or other HTML elements.

Basic Table Structure

To create a basic HTML table, you need the following elements:

  • <table>: The container for the entire table.
  • <tr>: Represents a table row. It contains one or more table cells (<td> or <th>).
  • <td>: Defines a standard table cell (data cell).
  • <th>: Defines a table header cell, typically used in the first row to label columns.

When rendered in a web browser, this code creates a table with two rows and two columns, where the first row contains header cells (in bold) and the subsequent rows contain data cells.
